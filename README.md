# My Profile Site

Static profile website for GitHub Pages.

## Local files

- `index.html` is the page content.
- `style.css` is the styling.

## Publish to GitHub

Create an empty GitHub repository named `my-profile-site`, then run:

```bash
git remote add origin https://github.com/jayakumar-ramalingam/my-profile-site.git
git branch -M main
git push -u origin main
```

Then enable GitHub Pages from the repository settings.
